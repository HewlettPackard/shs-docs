
# NVIDIA Collective Communications Library (NCCL) and AMD ROCm Collective Communications Library (RCCL)

Guidance for using the NVIDIA Collective Communications Library (NCCL) and AMD ROCm Collective Communications Library (RCCL) on HPE Slingshot NICs is available in the [shs-ccl-docs](https://github.com/HewlettPackard/shs-ccl-docs) repository.

The repository provides a customer-friendly guide for running applications that rely on NCCL or RCCL.
It also describes the engineering and protocol enhancements made to `aws-ofi-nccl` to enable robust, high-performance GPU-to-GPU collective communications over HPE Slingshot.
This GitHub repository replaces the previous NCCL/RCCL application note.

Customers are encouraged to use the repository's **Issues** tab to ask questions and provide feedback.
